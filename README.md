# PetLand 
El proyecto trata sobre reportes de animales callejeros que se pueden dar en adopción, localización de los mismos en un mapa y obviamente adoptarlos. Este proyecto es una iniciativa del grupo Desarrolladores .Net

## Temas a dominar al finalizar el proyecto
Durante el transcurso del desarrollo del proyecto se verán los siguientes temas esenciales para el desarrollo de una aplicación web
1) Autenticación en cliente y servidor
2) Arquitecturas limpias
3) Docker
4) Almacenamiento de ficheros en la nube
5) Inyecciones de dependencia
6) Envío de correos desde la plataforma

## Tecnologías
Para el desarrollo del sistema se tendrán en cuenta las siguientes tecnologías
1) PostgreSQL
2) .Net 7
3) Docker
4) Heroku CLI
5) Blazor WebAssembly
